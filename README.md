# Aiamblichus Tap

## How do I install these formulae?

`brew install aiamblichus/tap/<formula>`

Or `brew tap aiamblichus/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "aiamblichus/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
